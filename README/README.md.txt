# 📊 Projet BI Commercial – Système Décisionnel PME

## 🔍 Objectif du projet

Ce projet vise à concevoir et mettre en œuvre un système décisionnel (BI) simplifié pour une PME afin d'améliorer la **prise de décision commerciale** à partir des données de ventes.

L'objectif est de :
- Centraliser les données (produits, ventes, clients…)
- Mettre en place un entrepôt de données léger (modèle en étoile)
- Créer des tableaux de bord interactifs avec Power BI
- Fournir des KPIs métier clairs (CA, panier moyen, taux de fidélisation…)

## 📦 Contenu du dépôt

| Dossier                | Contenu                                                   |
|------------------------|------------------------------------------------------------|
| `00_Documents/`        | Cahier des charges, enjeux, présentation, journal de bord |
| `01_Donnees/`          | Fichiers CSV/Excel de simulation                           |
| `02_ETL/`              | Scripts Power Query ou Python pour nettoyage des données   |
| `03_Modelisation/`     | Modèle dimensionnel (SQL, schémas, dictionnaire)           |
| `04_Dashboards/`       | Fichiers `.pbix` Power BI                                  |
| `05_Scripts/`          | Scripts auxiliaires : DAX, SQL, automatisations            |

---

## 📊 Outils utilisés

- **Power BI Desktop** : création de tableaux de bord
- **Excel Power Query** : préparation et transformation des données
- **SQL Server Express** / **SQLite** : stockage local du modèle étoile
- **Git** : versionnage du projet
- **VS Code / Notepad++** : édition Markdown et scripts
